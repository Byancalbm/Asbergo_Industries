## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa [nome da empresa], realizado por [nome do responsável pelo projeto]. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- AWS Cost Explorer
- Monitoramento e análise de custos em tempo real
- A ferramenta foi adotada para permitir a visualização clara dos gastos por serviço, identificar picos de consumo e possibilitar ações rápidas de mitigação. Com isso, a empresa passou a ter maior controle financeiro e previsibilidade, permitindo ajustes estratégicos na alocação de recursos.

Etapa 2: 
- AWS Compute Optimizer
- Otimização de instâncias EC2, EBS, Lambda e outros recursos computacionais.
- O Compute Optimizer foi utilizado para analisar a performance das instâncias existentes, indicando onde havia recursos superdimensionados. Isso permitiu migrações para instâncias menores, descontinuação de recursos ociosos e ajustes automáticos de capacidade, resultando em economia imediata.

Etapa 3: 
- Amazon S3 Intelligent-Tiering
- Redução de custos de armazenamento com movimentação automática entre camadas.
- Arquivos armazenados em S3 foram configurados para o modo Intelligent-Tiering, que move automaticamente objetos entre classes mais baratas conforme o padrão de acesso. Isso elimina a necessidade de gerenciamento manual de ciclo de vida e reduz custos especialmente para dados com padrão de acesso imprevisível.
